"Modelo OSI"

Modelo de interconexion de sistemas abiertos, tambien llamado OSI es el modelo de red descriptivo propuesto para la organizacion internacional para la estandarizacion (iso). Normativaformada por siete capas que define las diferentes fases por las que pasan los datos para viajar de un dispositivo a otro.
